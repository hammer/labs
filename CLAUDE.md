# CLAUDE.md

## Deployment

- Site is hosted on Netlify at https://ai-lab-tracker.netlify.app/
- Netlify project name: `ai-lab-tracker`
- To link: `netlify link --name ai-lab-tracker`
- To deploy: `netlify deploy --prod`
- Build command: `npm run build` (Astro static site)

## Development

- Dev server: `npm run dev` (runs on http://localhost:4321)
- Install deps first: `npm install`
